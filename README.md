### 開発ツール###

* Ionic Framework
* Angularjs
* PhoneGap

### インストール方法###

1. #### Ionic Framework 
- Macの場合　`sudo npm install -g ionic `
- Windowsの場合　`npm install -g ionic `

2. #### PhoneGap
- Macの場合　`sudo npm install -g cordova `
- Windowsの場合　`npm install -g cordova `
